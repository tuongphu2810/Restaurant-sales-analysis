# Restaurant-sales-analysis

## Overview
This project analyzes restaurant sales data using Python, Pandas, and Matplotlib.  
The goal is to clean raw sales data, identify top-selling items, calculate total and daily revenue, and provide simple business insights.

## Skills Used
- Python
- Pandas
- Matplotlib
- Data Cleaning
- Data Analysis
- Data Visualization

## Project Objectives
- Clean and prepare restaurant sales data
- Calculate total revenue
- Find top-selling menu items
- Analyze daily revenue trends
- Compare revenue across categories
- Generate business recommendations

## Dataset Columns
The CSV file should include the following columns:

- `order_id`
- `date`
- `item_name`
- `category`
- `price`
- `quantity`
